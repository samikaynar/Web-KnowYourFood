# Know Your Food üçé

A simple web application that helps you understand how healthy your food is using AI.

## What It Does

- Enter any food product name
- Get a health score from 1 to 10
- See simple health advice
- Learn about better food choices

## How to Use

1. Open the website
2. Type a food name (like "chocolate" or "apple")
3. Click "Search"
4. Enter your OpenAI API key when asked
5. See the health analysis

## How It Works

- **Frontend**: HTML, CSS, JavaScript
- **AI**: OpenAI 
- **No backend needed** - runs entirely in your browser

## For Students

- HTML semantic elements (nav, header, section, article, footer)
- Modern CSS styling
